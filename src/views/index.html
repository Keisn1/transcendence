Welcome

Welcome to the Transcendence project. Our goal is to build a scalable, real-time web application using Node.js, Fastify, and modern frontend technologies.

Features

  • High-performance REST API with Fastify
  • Real-time updates with WebSockets
  • Modular architecture for easy maintenance
  • Static file serving and templating

Get Started

Clone the repository and install dependencies:

git clone https://github.com/Keisn1/transcendence
cd transcendence
make
make start

Visit http://localhost:3000 to see the app in action.
